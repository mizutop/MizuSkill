# Mizuof 工作室

> 完整产品生产流水线 ─ **16 角色 × 8 阶段 × 2 通道 × 300+ Skill**
> **用户只需一个想法 → 完整生产价值完成品**

## 核心理念

Mizuof 工作室通过 16 个角色化 AI 智能体、8 阶段审查流水线（含安全审计）、快速通道和 Mizu_skill 技能库，将模糊想法转化为安全、高质量的生产级成品。

## 流水线

**标准通道（8 阶段 + 安全审计）：**
```
构思 → 规划 → 设计 → 交互+视觉 → 开发TDD → 安全审计 → 质量审计 → 发布 → 复盘
```

**快速通道（Bug修复/热修复）：**
```
快速启动 → 开发 → 安全审计 → 质量审计 → 发布
```

---

## 快速开始

1. **提出想法** → Producer
2. **自动推进** → 每阶段质量评分通过后进入下一阶段
3. **角色执行** → 打开 `roles/` 下对应文件，按工作流步骤执行
4. **交付** → 经安全审计 → 质量审计 → 发布 → 复盘
